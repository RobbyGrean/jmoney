Thai User Guide

คู่มือการใช้งาน jmoney

หน้านี้ทำมาเพื่อให้ผู้ใช้ทั่วไปอ่านแล้วทำตามได้ทันที ตั้งแต่ดาวน์โหลด ติดตั้ง เปิดโปรแกรม ไปจนถึงการดูไฟล์ที่สร้างเสร็จและการแก้ปัญหาเบื้องต้น โดยตัวโปรแกรมนี้เป็น offline Windows app ที่ทำงานในเครื่องของคุณเอง

ใช้กับ Windows
ทำงานแบบออฟไลน์
สร้างเอกสาร Word จาก template
เหมาะกับผู้ใช้ทั่วไป
ดาวน์โหลดโปรแกรม ดาวน์โหลดคู่มือข้อความ

Section 1

วิธีติดตั้ง

1
แตกไฟล์ ReimbursementDocApp-Installer.zip ก่อน

หลังดาวน์โหลดเสร็จ ให้แตก zip ออกมาก่อน เพื่อให้เห็นไฟล์ติดตั้งด้านใน

2
เปิดไฟล์ ReimbursementDocApp-Setup.exe

นี่คือไฟล์ที่ใช้ติดตั้งโปรแกรมลงในเครื่อง Windows ของคุณ

3
รอให้โปรแกรมติดตั้งเสร็จ

ตัวติดตั้งจะคัดลอกไฟล์และตั้งค่าโฟลเดอร์ที่โปรแกรมต้องใช้ให้อัตโนมัติ

4
หลังติดตั้งเสร็จจะมีไอคอนบน Desktop

มองหาไอคอนชื่อ ReimbursementDocApp บนหน้า Desktop ได้เลย

5
เปิดโปรแกรมจากไอคอนบน Desktop

คุณสามารถเริ่มใช้งานได้ทันทีโดยไม่ต้องตั้งค่าอะไรซับซ้อนเพิ่ม

Section 2

ถ้า Windows ขึ้นหน้าฟ้าเตือน

Windows อาจขึ้นข้อความ “Windows protected your PC”

นี่เป็นคำเตือนปกติของ Windows สำหรับแอปที่ยังไม่เป็นที่รู้จักในวงกว้าง ไม่ได้แปลว่าเครื่องเสียหรือไวรัสเริ่มทำงานเองทันที

1
กด More info

ถ้ายังไม่เห็นปุ่มทำต่อ ให้กดปุ่มนี้ก่อน

2
มองหาปุ่ม Run anyway

หลังจากกด More info แล้ว Windows จะโชว์รายละเอียดเพิ่มและมีปุ่มนี้ให้

3
กด Run anyway ถ้าไฟล์มาจากหน้านี้จริง

ไฟล์ที่ควรเปิดคือ ReimbursementDocApp-Setup.exe หลังแตก zip แล้ว

ควรหยุดก่อนถ้ามีอะไรไม่ตรง

ถ้าชื่อไฟล์ไม่ตรง หรือคุณได้ไฟล์มาจากแหล่งอื่นที่ไม่ใช่หน้าแจกนี้ อย่าเพิ่งกดต่อ

Section 3

โปรแกรมถูกติดตั้งไว้ที่ไหน

โดยปกติโปรแกรมจะถูกติดตั้งไว้ที่ path นี้:

%LOCALAPPDATA%\ReimbursementDocApp

ตัวอย่าง path จริง:

C:\Users\ชื่อผู้ใช้\AppData\Local\ReimbursementDocApp
ไฟล์สำคัญที่อยู่ในโฟลเดอร์นี้

ReimbursementDocApp.exe คือโปรแกรมหลัก, Template คือโฟลเดอร์เก็บ Word template, output คือโฟลเดอร์เก็บเอกสารที่สร้างเสร็จ และ app_database.json คือข้อมูลที่ระบบใช้

ผู้ใช้ทั่วไปไม่จำเป็นต้องเข้า path นี้บ่อย

เพราะหลังติดตั้งจะมี shortcut บน Desktop และ Start Menu ให้แล้ว

Section 4

วิธีใช้งานแบบสั้นและชัด

1
เปิดโปรแกรม ReimbursementDocApp

เริ่มจากไอคอนบน Desktop หรือ Start Menu

2
ดูช่อง Template

ปกติโปรแกรมจะชี้ไปยังโฟลเดอร์ Template ให้อัตโนมัติอยู่แล้ว จึงไม่ต้องแก้ถ้าใช้งานตามปกติ

3
ดูช่อง Output

นี่คือที่เก็บไฟล์ Word ที่สร้างเสร็จแล้ว

4
เลือกเดือนส่งมอบและปี พ.ศ.

โปรแกรมจะคำนวณงวดและปีงบประมาณให้โดยอัตโนมัติ

5
เลือกเอกสารที่ต้องการสร้าง

ถ้ายังไม่แน่ใจ ให้เลือกทั้งหมดก่อนก็ได้

6
กรอกข้อมูลในแท็บต่าง ๆ ให้ครบ

ระบบถูกออกแบบให้กรอกข้อมูลชุดเดียวแล้วสร้างเอกสาร Word หลายไฟล์ต่อเนื่องได้

7
กดปุ่มสร้าง Word

โปรแกรมจะเปิดหน้าตรวจสอบอีกครั้งก่อนสร้างจริง

8
ถ้าข้อมูลถูกต้องให้กด OK

ถ้ายังต้องกลับไปแก้ ให้กด Cancel

9
หลังสร้างเสร็จ โปรแกรมจะถามว่าจะเปิดโฟลเดอร์ output หรือไม่

ถ้าต้องการดูไฟล์ทันที ให้ตอบเปิดโฟลเดอร์ได้เลย

Section 5

ไฟล์ที่สร้างเสร็จอยู่ตรงไหน

ปกติไฟล์เอกสารที่สร้างเสร็จจะอยู่ที่:

%LOCALAPPDATA%\ReimbursementDocApp\output

หรือดูจากช่อง Output ในโปรแกรมก็ได้

Section 6

ถ้าต้องการแก้ Template

ให้เข้าไปที่โฟลเดอร์นี้:

%LOCALAPPDATA%\ReimbursementDocApp\Template
แก้ไฟล์ Word `.docx` ได้โดยตรง

เหมาะสำหรับปรับข้อความหรือหน้าตาเอกสารบางจุดตามที่ต้องการ

อย่าลบ tag ในวงเล็บปีกกา

เช่น {ชื่อโรงเรียน} ถ้ายังต้องการให้โปรแกรมแทนค่าข้อมูลให้อัตโนมัติ

ปิดไฟล์ Word ก่อนกลับมาสร้างเอกสารใหม่

เพื่อป้องกันปัญหาไฟล์ถูกล็อกและเขียนทับไม่ได้

Section 7

ข้อควรระวัง

ปิดไฟล์ Word เดิมก่อนสร้างเอกสารใหม่

ถ้าไฟล์เดิมยังเปิดค้างอยู่ โปรแกรมอาจเขียนทับไฟล์นั้นไม่ได้

ตรวจเดือนส่งมอบและปี พ.ศ. ให้ถูกก่อนสร้าง

ค่าพวกนี้มีผลต่อข้อมูลที่โปรแกรมคำนวณและใส่ลงในเอกสาร

ถ้าโปรแกรมแจ้งว่าไม่พบ Template

ให้ตรวจว่ามีโฟลเดอร์ Template อยู่ในโฟลเดอร์ติดตั้งหรือไม่

ถ้าโปรแกรมเปิดได้แต่สร้างเอกสารไม่ออก

ให้ตรวจว่ามีโฟลเดอร์ output อยู่และใช้งานได้ตามปกติ

Section 8

ถอนการติดตั้ง และการขอความช่วยเหลือ

1
ถอนการติดตั้ง

เปิด Start Menu แล้วค้นหา ReimbursementDocApp จากนั้นเลือก Uninstall ReimbursementDocApp

2
ถ้าใช้งานไม่ได้

ควรแจ้งผู้ดูแลพร้อมข้อมูลว่าโปรแกรมขึ้นข้อความอะไร, ทำถึงขั้นตอนไหนแล้วมีปัญหา, ไฟล์ Word ที่ออกมาผิดตรงไหน และถ้ามีภาพหน้าจอให้แนบไปด้วย
