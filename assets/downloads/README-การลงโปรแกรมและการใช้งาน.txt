คู่มือใช้งานโปรแกรมออกเอกสารเบิกจ่าย
========================================

โปรแกรมนี้ใช้สำหรับกรอกข้อมูลครั้งเดียว แล้วสร้างเอกสาร Word สำหรับงานเบิกจ่ายจากแบบฟอร์มที่เตรียมไว้


1. วิธีติดตั้ง
--------------

1) แตกไฟล์ `ReimbursementDocApp-Installer.zip` ก่อน

2) เปิดไฟล์ `ReimbursementDocApp-Setup.exe`

3) รอสักครู่ โปรแกรมจะติดตั้งให้อัตโนมัติ

4) เมื่อติดตั้งเสร็จ จะมีไอคอน `ReimbursementDocApp` บน Desktop

5) เปิดโปรแกรมจากไอคอนบน Desktop ได้เลย


2. โปรแกรมถูกติดตั้งไว้ที่ไหน
-----------------------------

หลังติดตั้งแล้ว โปรแกรมจะอยู่ที่:

`%LOCALAPPDATA%\ReimbursementDocApp`

ตัวอย่าง path จริง:

`C:\Users\ชื่อผู้ใช้\AppData\Local\ReimbursementDocApp`

ภายในโฟลเดอร์นี้จะมี:

- `ReimbursementDocApp.exe` ตัวโปรแกรม
- `Template` โฟลเดอร์เก็บแบบฟอร์ม Word
- `output` โฟลเดอร์เก็บเอกสารที่สร้างเสร็จ
- `app_database.json` ข้อมูลที่โปรแกรมใช้

หมายเหตุ:
- ผู้ใช้ไม่จำเป็นต้องเปิด path นี้เพื่อใช้งานปกติ เพราะมี shortcut บน Desktop และ Start Menu ให้แล้ว
- ถ้าต้องการแก้แบบฟอร์ม Word ให้เข้าไปที่โฟลเดอร์ `Template`
- ถ้าต้องการดูไฟล์ที่สร้างแล้ว ให้เข้าไปที่โฟลเดอร์ `output`


3. วิธีใช้งาน
-------------

1) เปิดโปรแกรม `ReimbursementDocApp`

2) ตรวจดูช่อง `Template`
   โปรแกรมจะชี้ไปยังโฟลเดอร์ Template ในที่ติดตั้งอัตโนมัติ โดยปกติไม่ต้องแก้

3) ตรวจดูช่อง `Output`
   ช่องนี้คือที่เก็บไฟล์เอกสาร Word ที่สร้างเสร็จ

4) เลือกเดือนส่งมอบ และปี พ.ศ.
   โปรแกรมจะคำนวณงวดและปีงบประมาณให้อัตโนมัติ

5) เลือกเอกสารที่ต้องการสร้าง
   ถ้าไม่แน่ใจ ให้ติ๊กไว้ทั้งหมด

6) กรอกข้อมูลในแท็บต่าง ๆ ให้ครบ

7) กดปุ่ม `ตรวจสอบแล้วสร้าง Word`

8) โปรแกรมจะแสดงหน้าต่างให้ตรวจสอบอีกครั้ง
   ถ้าถูกต้องให้กด `OK`
   ถ้ายังต้องแก้ไขให้กด `Cancel`

9) เมื่อสร้างเสร็จ โปรแกรมจะถามว่าจะเปิดโฟลเดอร์ output หรือไม่


3.1 การบันทึก Template
------------------------

Template ใช้เก็บข้อมูลประจำของผู้รับจ้างหรือโรงเรียน เพื่อไม่ต้องกรอกใหม่ทุกครั้ง

1) กรอกข้อมูลประจำให้ครบ เช่น ผู้รับจ้าง โรงเรียน ตำแหน่ง ผู้ลงนาม และคณะกรรมการ

2) กดปุ่ม `บันทึกเป็น Template`

3) เลือกบันทึกเป็นรายการใหม่ แล้วตั้งชื่อและหมายเหตุให้ค้นหาได้ง่าย
   เช่น ชื่อผู้รับจ้าง หรือชื่อโรงเรียน

4) หากแก้ไขข้อมูลประจำของ Template เดิม ให้เลือกบันทึกทับ Template เดิม
   ตรวจชื่อ Template ให้ถูกก่อนยืนยัน


3.2 จัดการ Template และโหลดเข้าฟอร์ม
--------------------------------------

1) กดปุ่ม `จัดการ Template`

2) เลือกรายการ Template ทางด้านซ้าย แล้วดูรายละเอียดทางด้านขวา

3) กด `โหลดเข้าฟอร์ม` เพื่อดึงข้อมูลของรายการนั้นกลับมาแก้ไขหรือสร้างเอกสาร
   วิธีนี้ไม่เลื่อนเดือนอัตโนมัติ ให้ตรวจและเลือกเดือนส่งมอบเอง

4) ใช้ปุ่ม `แก้ไข/หมายเหตุ` เมื่อต้องการเปลี่ยนชื่อหรือหมายเหตุ

5) ใช้ปุ่ม `ลบ` เฉพาะ Template ที่ไม่ใช้แล้ว การลบไม่กระทบไฟล์ Word ที่สร้างเสร็จแล้ว


3.3 เหมือนเดิม! แค่เปลี่ยนเดือน
---------------------------------

1) กดปุ่ม `เหมือนเดิม! แค่เปลี่ยนเดือน!` แล้วเลือก Template ที่ต้องการใช้ต่อ

2) ระบบจะอ้างอิงงวดที่สร้างสำเร็จล่าสุดของ Template นั้น และเสนอเดือนถัดไป
   จึงไม่ต้องเดาว่าผู้ใช้กำลังเบิกเดือนใดจากวันที่ที่เปิดโปรแกรม

3) อ่านหน้าต่างยืนยันเดือนและปี พ.ศ. หากถูกต้องให้กดยืนยัน
   ระบบจะโหลดข้อมูลเดิมและตั้งเดือนส่งมอบเป็นงวดถัดไป

4) ตรวจเดือนส่งมอบ ปีงบประมาณ และข้อมูลบนฟอร์มก่อนกดสร้าง Word ทุกครั้ง

5) เมื่อสร้าง Word สำเร็จ ระบบจึงบันทึกงวดล่าสุดใหม่ของ Template นั้น
   หากสร้างเอกสารย้อนหลัง ระบบจะไม่ถอยงวดล่าสุดกลับไป


4. ไฟล์ที่สร้างเสร็จอยู่ที่ไหน
------------------------------

โดยปกติไฟล์ที่สร้างเสร็จจะอยู่ที่:

`%LOCALAPPDATA%\ReimbursementDocApp\output`

ตัวอย่าง path จริง:

`C:\Users\ชื่อผู้ใช้\AppData\Local\ReimbursementDocApp\output`

หรือดูจากช่อง `Output` ในหน้าโปรแกรมก็ได้


5. ถ้าต้องการแก้ Template
--------------------------

ให้เข้าไปที่โฟลเดอร์:

`%LOCALAPPDATA%\ReimbursementDocApp\Template`

แล้วแก้ไฟล์ Word `.docx` ได้เลย

ข้อแนะนำ:
- แก้เฉพาะข้อความหรือรูปแบบที่ต้องการ
- อย่าลบ tag ในวงเล็บปีกกา เช่น `{ชื่อโรงเรียน}` ถ้ายังต้องการให้โปรแกรมแทนค่าอัตโนมัติ
- ถ้าแก้ template แล้ว ควรปิดไฟล์ Word ก่อนกลับมาสั่งสร้างเอกสาร


6. ข้อควรระวัง
---------------

1) ก่อนกดสร้างเอกสาร ควรปิดไฟล์ Word ที่เคยเปิดค้างไว้ก่อน
   ถ้าเปิดไฟล์เดิมค้างไว้ โปรแกรมอาจเขียนทับไฟล์นั้นไม่ได้

2) ตรวจเดือนส่งมอบและปี พ.ศ. ให้ถูกต้องก่อนสร้าง

3) ถ้าโปรแกรมแจ้งว่าไม่พบ Template
   ให้ตรวจว่าโฟลเดอร์ `Template` ยังอยู่ในโฟลเดอร์ติดตั้งหรือไม่

4) ถ้าโปรแกรมเปิดได้ แต่เอกสารสร้างไม่ออก
   ให้ตรวจว่าโฟลเดอร์ `output` ยังอยู่หรือไม่


7. การถอนการติดตั้ง
-------------------

วิธีถอนการติดตั้ง:

1) เปิด Start Menu

2) ค้นหา `ReimbursementDocApp`

3) เลือก `Uninstall ReimbursementDocApp`


8. หากใช้งานไม่ได้
------------------

ให้แจ้งผู้ดูแลพร้อมข้อมูลเหล่านี้:

1) โปรแกรมแจ้งว่าอะไร

2) ทำถึงขั้นตอนไหนแล้วเกิดปัญหา

3) ไฟล์ Word ที่สร้างออกมาผิดตรงไหน

4) ถ้ามีภาพหน้าจอ ให้ส่งภาพหน้าจอมาด้วย
